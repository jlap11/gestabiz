# Gestabiz Mobile Assets

Los siguientes archivos deben ser agregados a esta carpeta:

- **icon.png**: Icono de la app (1024x1024px)
- **adaptive-icon.png**: Icono adaptativo para Android (1024x1024px)
- **splash.png**: Pantalla de splash (1284x2778px)
- **favicon.png**: Favicon para web (48x48px)

Por ahora, puedes usar imágenes placeholder o logos simples.

## Generación rápida
Puedes usar herramientas como:
- https://www.appicon.co/ (generador de iconos)
- https://easyappicon.com/ (generador para todas las plataformas)
- Canva (crear diseños personalizados)

## Notas
- Los archivos deben estar en formato PNG
- El icono debe tener fondo transparente o color sólido
- La splash screen debe ser visualmente simple y rápida de cargar
